### The Automation Criteria (The "Should I?" Test)
Before automating, ask:
1.  **Is it repetitive?** (Does it run every sprint?)
2.  **Is it stable?** (Is the UI finished, or is it still shifting?)
3.  **Is it high volume?** (Does it require 100 different data combinations?)
