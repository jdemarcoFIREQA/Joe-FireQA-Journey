# Joe-FireQA-Journey
A learning, documentation, and portfolio repository. A journey to improve software QA skills including test cases, bug reports, documentation, and automation practice. 

QA Skill Development Roadmap
This repo documents my 30-day structured learning plan to strengthen my skills as a software quality assurance engineer while applying them to my current job. 

Includes
Daily/weekly progress logs
Test cases & templates
Bug reports
Notes from exploratory testing
Light automation practice (Cypress/Playwright)
A GitHub project board to track projects. 

Goals for the first 30 days. 
* Strengthen core QA thinking and test case design
* Improve exploratory testing habits
* Learn better bug reporting & communication
* Build foundational automation comfort
* create a public QA porfolio
